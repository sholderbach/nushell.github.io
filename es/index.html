Nushell

Un nuevo tipo de shell.

Pipelines potentes para controlar tu sistema

Pipelines permiten trabajar con tu sistema como nunca antes. Tienes el control del sistema, listo para tu siguiente comando.

Todo es data.

En lugar de tener la necesidad de recordar todos los parámetros de todos los comandos, podemos usar los mismos, independientemente de su procedencia.

Plugins

Nu no puede incluir todo lo que quieras hacer con él, por lo que puedes extenderlo usando su sistema fuerte de plugins.

# Instalación rápida

# Ultima versión

Instala la última versión en la pagina de releasesopen in new window

Desde la fuente en crates.ioopen in new window:

$ cargo install nu

# Instalación a través de resolvedores de paquetes

En Homebrewopen in new window:

$ brew install nushell

En Scoopopen in new window:

$ scoop install nu

# Inicia la shell

$ nu
